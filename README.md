# Wi-Fi Quick Refresh (Windows)

Fixes:
- "Connected but no internet"
- eduroam / GSB Wi-Fi freezes
- Captive portal timeout issues

## Usage
Run as Administrator:

## Why?
Some networks silently drop DHCP / session without disconnecting Wi-Fi.
This script refreshes the session without rebooting or toggling Wi-Fi.

Tested on:
- Windows 10 / 11
- eduroam
- GSB Wi-Fi
